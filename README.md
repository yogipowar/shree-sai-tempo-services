# Shree Sai Tempo Services

A modern, premium, responsive website for **Shree Sai Tempo Services** — a professional transportation and logistics company based in Pune, Maharashtra, India.

## Tech Stack

- **React 19** — Component-based UI
- **Vite** — Fast development and build tooling
- **Plain CSS** — Modular, mobile-first styles with CSS custom properties

## Features

- Sticky transparent navbar with blur effect
- Hero section with trust badges and animated visuals
- Animated statistics counters
- Service cards with hover lift effects
- Fleet showcase with booking CTAs
- Interactive service areas map
- Glassmorphism testimonial carousel
- FAQ accordion
- Contact form with Google Maps embed
- Fade-up scroll animations
- Fully responsive (mobile-first)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI (Navbar, Footer, Button, SectionHeading)
├── sections/       # Page sections (Hero, Services, Fleet, etc.)
├── hooks/          # Custom hooks (scroll animation, counters, navbar)
├── data/           # Content data (services, fleet, FAQ, etc.)
├── styles/         # Global CSS and design tokens
├── App.jsx         # Main app layout
└── main.jsx        # Entry point
```

## Customization

- **Colors & fonts**: Edit CSS variables in `src/styles/globals.css`
- **Content**: Update data in `src/data/content.js`
- **Contact info**: Change phone, email, and address in `src/data/content.js`
- **Images**: Replace Unsplash URLs with your own vehicle and team photos
